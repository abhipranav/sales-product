# North-Star Metrics

## Primary outcomes

- Rep admin hours per week (target: -30% to -40% from baseline)
- Follow-up SLA compliance rate
- Stage-to-stage conversion from discovery to proposal

## Secondary outcomes

- Multithreaded deal rate
- Sales cycle duration
- Forecast confidence calibration
- AI recommendation acceptance rate
- Time from meeting end to first high-quality next action
- Approval queue turnaround time
